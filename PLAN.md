# der包个人网页设计计划

## 项目概述
重新设计 der包 个人网页，使用电影感编辑风格，以山景日落图片作为视觉核心。

- 重构导航栏结构
- 重新设计 Hero 区域（全屏背景图片）
- 优化关于我区块布局
- 重新设计作品集展示
- 更新联系区块样式
- 添加装饰性背景文字

### 2. CSS 样式设计 ✅
- 定义设计系统变量（暗色主题、金色点缀）

- 预加载器动画
- Hero 区域电影感效果
  - 图片缩放入场动画

## 设计特点

### 视觉风格
- **色彩**：深色背景 (#0a0a0a) + 金色点缀 (#c9a962)
- **字体**：ZCOOL XiaoWei（标题）+ 系统字体（正文）
- **氛围**：电影感、编辑风格、优雅简约

### 交互细节
- 自定义光标
- 平滑过渡动画
- 滚动视差效果
- 作品筛选动画
